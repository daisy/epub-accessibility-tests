extdescription-010 Detection and Utilization of HTML Details and Summary elements Below Image

Check if you can use HTML details and summary successfully and read the extended description contained in it.

This test passes if all the following steps are successful:

  1. The HTML details/summary elements are reported. (For example, there is an interactive element like button, link etc. that you can activate.)
  2. You can expand and collapse the details element containing the extended description. (For example, you can activate the interactive element to expand or collapse the extended description.)
  3. When expanded, the extended description should appear immediately after the activating element. When you collapse it, you should remain at the same position.
  4. When expanded, you should be able to read the extended description including being able to explore the HTML table within the description.
