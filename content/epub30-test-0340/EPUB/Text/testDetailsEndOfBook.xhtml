Extended descriptions using links to end of chapter

Notes about techniques

In this chapter there are two examples. The extended descriptions are placed at the end of the book. It is expected that this approach will appeal to publishers that do not want the descriptions to impact the appearance of the content. The approaches have the lowest impact on the visual layout. The bar chart has a text link and the peacock has an image as a link. A link back to the image is provided to take the reader back to the same location where they left off reading in the primary flow of content.

This test passes if all the following steps are successful:

  • The link "Follow for extended description" is reported, and on activating, it takes the focus to extended description at end of this book. The extended description is placed under the heading "Description of Bar Chart"

  • On activating the link "Navigate back to Bar Chart image" below the extended description, it takes the focus back to original position in this chapter.
