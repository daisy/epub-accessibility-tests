Advanced Read Aloud Tests

The tests given below are for evaluating the Read Aloud feature within the reading system. Do not perform these reading tests using a screen reader.

AdvancedReadAloud-200 Text to Speech Changes Languages Automatically

Read aloud should automatically change the TTS speaking language when it encounters a language change in the XHTML. This test contains English, German, French, and Italian.

Using the read aloud feature listed to the phrases below. The test passes if the TTS language changes as it encounters the phrases in the different languages. If this does not happen, check that you have the relevant voices available on your device before marking as a fail.

Indicate Pass or Fail.

Language Change Test

In English, "Good morning my darling."
In German, "Guten Morgen, mein Liebling."
In French, "Bonjour ma chérie."
In Italian, "Buongiorno amore mio."

End of language change testing.

AdvancedReadAloud-400 Image alternate text reading

Ensure that the alternate text for the image can be announced.

If the read aloud announces the alternate text of the image above then the test passes. The alternate text should read as "Moby Dick with a sailor in his mouth and several with harpoons hanging on him and their ship in the background."

Indicate Pass or Fail.

AdvancedReadAloud-600 The read aloud emphasis or highlight colour can be changed

Activate Read Aloud feature as described in the previous test. Change the colour of emphasis or visual highlight of the text being read aloud using any feature available in the reading system or the operating system. The user should have a choice of different colours for text highlighting and/or the option to disable the visual highlighting.

Indicate Pass or Fail.

AdvancedReadAloud-800 MathML Reading

The block and inline math expressions below are in the MathML format. We want to determine that the read aloud announces the math correctly.

Check if the read aloud announces the math expressions correctly in both samples. The test fails if the math is ignored, or you cannot read/hear the equation containing an equals sign and the presence of a fraction contained within. Indicate Pass or Fail.

MathML Block Test

Block Math Equation will appear between the following two lines.

Text before block math. y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 ) Text after block math.

MathML Inline Test

Inline Math Equation will appear between the following two lines.

Text before inline math. y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 ) Text after inline math.

Please describe your reading experience in the notes.
