Advanced Read Aloud Tests

The tests given below can be conducted using a Read Aloud feature offered within the reading system, web browser, a third-party tool, or via an Operating System feature. Do not perform these reading tests using a screen reader designed for blind users.

AdvancedReadAloud-200 Text to Speech Changes Languages Automatically

Read aloud should automatically change the TTS speaking language when it encounters a language change in the XHTML. This test contains English, German, French, and Italian.

Using the read aloud feature listen to the phrases below. The test passes if the TTS language changes as it encounters the phrases in the different languages. If this does not happen, check that you have the relevant voices available on your device before marking as a fail.

Indicate Pass or Fail.

Language Change Test

In English, "Good morning my darling."
In German, "Guten Morgen, mein Liebling."
In French, "Bonjour ma chérie."
In Italian, "Buongiorno amore mio."

End of language change testing.

AdvancedReadAloud-400 Image alternate text reading is optional

Ensure that the user can choose whether to hear the announcement of the alternate text for the image.

Configure the read aloud feature so alternate (alt) text of images is announced. Check with the image above, this should read as "Moby Dick with a sailor in his mouth and several with harpoons hanging on him and their ship in the background."

Next, configure the read aloud feature so that alternate text is not announced. Check that the alternate text for the image is ignored.

The test passes if the alternative text is announced with the feature enabled, and ignored if the feature is disabled. Indicate Pass or Fail.

AdvancedReadAloud-600 The read aloud emphasis or highlight colour can be changed

Change the emphasis or visual highlight color of the text being read aloud using any feature available in the reading system or the operating system. The user should have a choice of different colours for text highlighting and/or the option to disable the visual highlighting.

Indicate Pass or Fail.

AdvancedReadAloud-800 MathML Reading

The block and inline math expressions below are in the MathML format. This test is to determine that the read aloud feature announces the math expressions correctly.

Check if the read aloud announces the math expressions correctly in both samples. The test fails if the math is ignored, or you cannot read/hear the equation containing an equals sign and the presence of a fraction contained within. Indicate Pass or Fail.

MathML Block Test

Block Math Equation will appear between the following two lines.

Text before block math. y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 ) Text after block math.

MathML Inline Test

Inline Math Equation will appear between the following two lines.

Text before inline math. y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 ) Text after inline math.
