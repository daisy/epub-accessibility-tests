Reading History Navigation Tests

In this section we will be examining the go-back or history function in reading systems. It is the task of the reading system to keep track, as best it can, of your reading location. When the reader navigates to another location, the reading system should store the reading position to enable the user to return to that location.

When reporting your results in the notes field, please explain your experience. You may be in scrolling mode, which makes it mor difficult for the reading system to know your reading position. It may be that you are following a link, which means the reading system should be able to identify the exact location to return to. Other functions, like go to page or starting a search can be initiated anywhere in the content. In this case the reading system should track the last known position, probably the first element in the viewport. If using a screen reader with a virtual buffer, there may be mechanisms to tell the reading system where you are. These techniques should be tested and reported in the notes.

history-010 Four single follow link and return tests

This test involves four parts. It tests going to a footnote, and then returning, and the same process for an end note, a glossary entry, and a bibliographic reference. A reading system may have a pop-up for footnotes and/or endnotes, and the way to return may be to escape or dismiss. Note if this behavior is used.

Steps

  1. Follow the link to the footnote. 1.
  2. Return here using the go back function.
  3. Follow the link to the end note. 2.
  4. Return using the go back function.
  5. Follow the link to the glossary reference. database.
  6. Return to this location using the go back function.
  7. Follow the link to the bibliographic reference. A Collection of Important Writings.
  8. Return to this location using the go back function.

If all four of the links through the book are traversed and the reading system's go back function or dismiss pop-up returns to this location each time, the test passes. Note any issues encountered.

Indicate Pass or Fail.

Filler paragraph. The history of whaling in Maui is a significant chapter in the island's past, primarily centered around the town of Lahaina. During the 19th century, particularly between the 1820s and 1860s, Lahaina became one of the busiest and most important whaling ports in the Pacific. Hundreds of whaling ships would anchor in its waters, seeking to replenish supplies and offload their valuable cargo of whale oil and baleen. This lucrative industry brought both prosperity and profound cultural shifts to Maui. While it provided economic opportunities, it also led to clashes between the rough-and-tumble whalers and the more conservative Hawaiian missionaries and chiefs. The decline of whaling in the latter half of the 19th century was due to several factors, including the discovery of petroleum, the decimation of whale populations, and the American Civil War, ultimately leading to the industry's demise in Maui and marking a transition for the island's economy.

This test determines if the reading system's search function supports the go back function. Ideally, the go back function avoids taking the user to the search box, i.e. you do the search and return directly to this location.
