MathML Read-Aloud Tests

math-150 MathML Read-Aloud

This test should be done by using "read aloud" feature of the reading system.

The test passes if the following is true:

Either the mathML is spoken correctly or the alt txt is spoken i.e.: Mathml-alt-text followed by the math equation: y minus y sub 1 equals StartFraction y sub 2 minus y sub 1 Over x sub 2 minus x sub 1 EndFraction left-parenthesis x minus x sub 1 right-parenthesis

The test fails if you do not hear any math spoken or if you don't hear the math equation containing a fraction and subscripts.

Test 150

Math Equation will appear between the following two lines.

y − y 1 = y 2 − y 1 x 2 − x 1 ( x − x 1 )

Additional Notes for Test 150

Please provide notes about what you heard. For example, did you hear the prefix "Mathml-alt-text”. If you didn't hear that prefix before the spoken math please provide the math expression you hear spoken in your notes.

Please indicate in your notes if you know that this Reading System uses MathJax and ideally the version number of MathJax if possible.
